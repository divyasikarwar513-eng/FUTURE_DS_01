# # FUTURE_DS_01 – Business Sales Performance Analytics

This project analyzes business sales performance using Microsoft Excel.

## Dashboard Features
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- KPI Metrics
- Business Insights

## Tools Used
- Microsoft Excel
- PivotTables
- Charts

## Key Insights
- West region generated the highest sales revenue.
- Technology category produced the highest profit.
- November and December showed strong sales performance.
